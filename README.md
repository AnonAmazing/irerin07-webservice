# 스프링 부트와 AWS로 혼자 구현하는 웹 서비스 

## 책을 따라가면서 게시판을 만들며 여러 기술들을 가볍게 배울 수 있었다. 깊이있게 들어가며 공부하는 것은 오롯이 나의 몫. 화이팅!

## 사용기술
- Java8
- SpringBoot
- AWS(EC2, RDS, IAM, S3, CodeDeploy)
- Travis-ci
- Mustache

- Admin 권한 추가(모든 게시글 수정 삭제 가능)
- 댓글
- 파일 업드로

- 일반 회원가입 추가
